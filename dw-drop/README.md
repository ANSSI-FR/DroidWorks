# dw-drop

## Dependencies

To run embedded apksigner, the `java` program must be installed and set to the path.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
cargo tauri dev
```

### Compiles and minifies for production
```
cargo tauri build
```

### Lints and fixes files
```
npm run lint
```
